操纵蛇请使用awsd（仅支持小写）
其他操作核心思想为：滚轮切换选项卡，左键确认，右键返回，Boss战内空格打开策略


Mode1—3分别对应基础要求中的三个模式，游戏开始时，按回车正式开始；
游戏结束时，自动记录本次数据；
游戏过程中按q退出游戏，不记录数据；
其中，模式二很难达到结束条件，故按q退出时会记录数据


故事模式请按回车推进剧情，不记录任何游戏数据
内含Boss战，游戏过程中给出的策略提示足够通关，如果您想完整体验，请略过本段以下介绍内容。
Boss战操作：移动控制与上述相同，随时间积累或吃到食物，会在左侧显示有ATB能量条，每咱满1ATB能量条会亮起提示，最多可攒2ATB。战斗过程随时可按下空格键，进入子弹时间，使用鼠标滚轮+左键+右键选择消耗ATB的策略。选择“洞察”将获取敌人弱点信息，选择“恢复”将回复生命值，选择其余四项将对Boss发动魔法攻击。Boss拥有护盾，护盾属性可根据Boss字颜色判断，会随机变化。Boss拥有两个技能，“护盾扩散”，在场上生成同护盾属性的食物；“分裂墙壁”，在场上随机生成墙体。技能发动前，有闪烁提示


Rank排行榜显示已有的9条最高分数据，界面内按回车退出，按q查询某用户名的游戏记录
请注意：查询用户名记录时，只允许查询总榜前九条的记录，且显示出的位置不变。


排行榜增删改查的要求分别实现为：自动记录数据（增），不够前九名的数据被清理（删），获取高分可将别人挤出排行榜（改），按用户名查询（查）
